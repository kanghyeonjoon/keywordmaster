# 🎬 유튜브 기획안·대본 자동 생성 시스템

유튜브 운영 대행 업무에서 매번 수작업으로 만들던 **영상 기획안과 대본**을 자동으로 생성하는 시스템입니다.

## 어떻게 동작하나요?

```
[script-generator.html 폼 입력]
        ↓ 전송
[Make.com 웹훅 수신 → 즉시 "접수 완료" 응답]
        ↓
[Claude AI가 기획안 + 대본 생성 (1~3분)]
        ↓
[입력한 이메일로 결과 도착 📧]
        ↓
[Google Docs 문서로도 자동 저장 (내 드라이브) 📄]
```

이메일 한 통에 아래가 모두 담겨 옵니다.

**📋 기획안**
1. 제목 후보 5개 (클릭 유도 포인트 설명 포함)
2. 오프닝 훅 전략 (롱폼: 첫 15초 / 숏폼: 첫 3초)
3. 타깃 시청자 분석
4. 영상 구성안 (섹션별 시간 배분 표)
5. 썸네일 문구 후보 3개
6. 추천 태그·키워드 10개
7. 예상 이탈 포인트와 대응 방법

**🎬 대본**
- 롱폼: 인트로 훅 → 섹션별 풀스크립트(구어체) → 아웃트로 CTA, (화면/자막 지시) 포함
- 숏폼: 0~3초 훅 → 본론 → CTA, 60초 분량, 자막 줄 단위

## 사용 방법

1. `script-generator.html` 파일을 브라우저로 엽니다 (더블클릭 또는 GitHub Pages 주소 접속)
2. 채널명 · 분야 · 형식(롱폼/숏폼) · 주제를 입력합니다
3. **벤치마킹 도입부 예시**(선택): 잘 터진 영상들의 도입부 문장을 한 줄에 하나씩 붙여넣으면,
   AI가 공통 구조를 분석해 "도입부 공식"을 만들고 그 공식을 적용한 도입부 3버전 + 같은 패턴의 제목/썸네일 문구를 생성합니다
4. "생성하기" 클릭 → 1~3분 뒤 이메일 확인 (같은 내용이 Google Docs 문서로도 내 드라이브에 저장됩니다)

## 🎣 도입부 프리셋 관리 (hooks.js)

모아둔 도입부를 매번 붙여넣지 않아도 됩니다. `hooks.js` 파일에 메모장처럼 저장해두면
폼에 **"도입부 프리셋" 드롭다운**이 생기고, 선택하면 자동으로 입력됩니다.

**편집 방법**: GitHub에서 `hooks.js` 열기 → 연필(✏️) 아이콘 → 수정 → Commit

```
[채널A 스타일]
- 이거 모르면 월급 300 그냥 버리는 겁니다
- 제가 3년간 1000명을 상담하며 알게 된 사실인데요

[병원 채널용]
- 환자분들이 진료실에서 가장 많이 묻는 질문이 있습니다
```

- `[이름]` 줄로 프리셋을 구분 (채널별·스타일별로 여러 개 가능)
- 한 줄에 도입부 하나씩
- 백틱(`) 문자만 사용 금지

## 관리 방법 (Make.com)

- 시나리오 이름: **"영상 기획안·대본 자동 생성 (폼 → Claude → Gmail)"**
- 켜기/끄기: [Make.com](https://eu2.make.com) → Scenarios → 해당 시나리오 ON/OFF
- **Claude API 키 교체**: Make → 우측 상단 프로필 → Resources/Keychains → `Anthropic` 키(ID 200206) 값 수정
- **메일이 안 올 때**: Make → 해당 시나리오 → History 탭에서 실패한 실행을 클릭하면 어느 단계에서 멈췄는지 보입니다
  - Claude 호출(HTTP) 단계 실패 → API 키 만료 또는 크레딧 부족일 가능성이 큼
  - Gmail 단계 실패 → Google 연결 재인증 필요
- **메일은 오는데 Google Docs 문서가 안 만들어질 때**: Google 연결에 문서 권한이 없는 경우입니다.
  Make → 시나리오 편집 → 마지막 Google Docs 모듈 클릭 → 연결(Connection)에서 "Add" 또는 재인증 후 저장.
  (Docs 저장은 실패해도 무시되도록 설정되어 있어 메일 발송에는 영향이 없습니다)

## 파일 구성

| 파일 | 용도 |
|---|---|
| `script-generator.html` | 기획안·대본 생성 입력 폼 (이 시스템의 입구) |
| `hooks.js` | 도입부 프리셋 모음 (메모장처럼 편집) |
| `make-blueprint.json` | Make 시나리오 백업본 (수동 복원용) |
| `index.html` | 환자 심리 키워드 분석기 (기존 도구, 별개) |
